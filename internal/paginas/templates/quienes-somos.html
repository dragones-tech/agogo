Quiénes somos

Agogo es un proyecto que demuestra que se puede construir un sitio web seguro, rápido y mantenible con Go puro: la biblioteca estándar, plantillas para SEO y una API JSON, sin frameworks ni dependencias de más.

Cada pieza es código nuestro: lo leemos, lo entendemos y lo arreglamos sin depender de cajas negras externas.

Panel externo (HTML sobre el cable)

Al pulsar el botón, el servidor llama a la API de GitHub (la que normalmente usas con un token para más rate-limit o repos privados), trae las estadísticas del repo y devuelve el HTML ya armado. El token nunca llega al navegador; el cliente solo hace innerHTML — sin JSON ni framework en la página. Es HTML over the wire. (Aquí la API va simulada; en producción apuntas a la real.)
